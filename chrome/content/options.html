Preferências

Todas as páginas

Faz com que o texto “Pesquisa” não desapareça automaticamente ao clicar no campo, somente após começar a digitar. Quando o texto digitado é excluído, a palavra “Pesquisa” aparece novamente.

Telas com lista de localizadores

Nas telas em que aparece uma lista de localizadores, torna os localizadores vazios menos destacados, para que a atenção seja naturalmente direcionada aos demais.

Telas com lista de processos

Quando uma tabela com vários processos possui poucas colunas, ao usar um monitor grande, as informações podem parecer “esticadas”. Selecionando esta opção, as tabelas ocupam apenas o espaço necessário para exibir as informações disponíveis.

Telas com documentos

Modifica a formatação do último link de documento clicado, para facilitar sua identificação entre os demais documentos do processo. Útil quando há vários documentos com nomes semelhantes em sequência, no mesmo evento.

Controla as janelas de documentos abertas em determinado processo, permitindo que sejam fechadas ao sair da tela do processo ou quando o usuário achar conveniente.

Tela de seleção de perfil

Após o login, quando o usuário possui mais de um perfil no eproc, é mostrada uma tela com uma lista dos perfis para que possa escolher qual deseja utilizar. Esta preferência centraliza a lista, que geralmente aparece deslocada para a direita.

Tela do processo

Cria uma área abaixo da lista de minutas onde é possível carregar os documentos do Gedpro vinculados ao processo.

Exibe ícones nos botões do menu “Ações”, facilitando a identificação da opção desejada entre as diversas disponíveis.
Atenção: esta preferência somente funciona se a configuração do eproc “Botões no menu ‘Ações’” estiver ativada.

Alguns eventos, como “Intimação Eletrônica - Confirmada” e “Decurso de Prazo”, informam o número do evento a que se referem. Porém, é preciso consultar o evento original para saber qual parte estava sendo intimada/citada. Com esta preferência, o tipo (autor, réu, interessado, etc.) e nome da parte aparecem também no evento referido.
